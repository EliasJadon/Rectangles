# Rectangles POC
A POC project mixing [libigl](https://github.com/libigl/libigl/) and
[TinyAD](https://github.com/patr-schm/TinyAD).

## Objective functions
1. Symmetric Dirichlet
2. Rectangle
3. Pin vertices

## Compile

Compile this project using the standard cmake routine:

    mkdir build
    cd build
    cmake ..
    make
